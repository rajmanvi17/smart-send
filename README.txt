# WhatsApp Automation using PyWhatKit

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Edit `contacts.csv` with your contacts and message details.

3. Run:
   ```bash
   python whatsapp_automation.py
   ```

4. Scan QR code on WhatsApp Web when prompted.

## Notes
- Ensure your internet and WhatsApp Web stay active while sending.
- Phone numbers must include country code.
- Avoid spamming — use responsibly.
